# Frontend Philosophy

1. Use Tailwind utility classes; avoid arbitrary values.
2. Prefer shadcn primitives over custom components.
3. Keep components composable and props explicit.
4. Accessibility first: labels, focus states, color contrast.
5. Responsive by default, not as an afterthought.
